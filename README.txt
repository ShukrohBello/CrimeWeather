HOW TO RUN
________________
Make sure the three .csv files, crime_reports, weather_2015, and weather_2016
are in the same directory as your jupyter notebook. From there you should
be able to easily run the code by simply running each cell from beginning to
end, making sure to pip install and import the necessary libraries in the
first cell of the notebook.
